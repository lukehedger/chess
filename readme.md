# Unlease Chess

## Setup

### Global dependencies
> [`npm-run`](https://www.npmjs.com/package/npm-run) is used to run the local `webpack` executable, rather than depend on a global package

```bash
npm i -g npm-run
```

### Local dependencies
```bash
npm i
```

## Tasks

### Develop
```bash
npm start
```

### Test
```bash
npm test
```

### Release
```bash
npm run build
```

### Deploy
```bash
npm run deploy
```
